# 📋 Morning Shift Report
**Infozillion Teletech Bd Ltd · Service Assurance**

A professional, offline-capable shift reporting tool for the Service Assurance team.

---

## 🚀 Features

| Feature | Description |
|---|---|
| 🌙 Dark Mode | Toggle light/dark theme, saved across sessions |
| 💾 Auto-save | Draft automatically saved to browser (never lose data) |
| 🕓 History | Last 30 reports saved, copyable anytime |
| 📂 Collapse | All sections collapse/expand for easy navigation |
| 📋 Copy | One-click WhatsApp-ready report generation |
| 🖨 Print / PDF | Clean print layout for PDF export |
| 📱 Mobile | Fully responsive for phones and tablets |


---

## 🌐 Deploy on GitHub Pages

1. Create a new GitHub repository (e.g. `shift-report`)
2. Upload all 3 files: `index.html`, `style.css`, `app.js`
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch → `/ (root)`
5. Click **Save**
6. Your report will be live at:
   ```
   https://<your-username>.github.io/shift-report/
   ```

---

## 💡 Usage

1. Open the page in any browser
2. Fill in the form sections (data auto-saves as you type)
3. Click **"Copy WhatsApp Message"**
4. Paste directly into your WhatsApp group

---

## 🔒 Privacy

All data is stored **locally in the browser** using `localStorage`.  
No data is sent to any server. Works fully offline.
